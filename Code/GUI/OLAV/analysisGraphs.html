OPENSTACK LOG ANALYSIS AND VISUALIZATION

Provides Analysis and a clean Visualization of Openstack Logs

ERROR LOCATOR

Provides a user friendly visualization of the occurence of errors with a comparison of a successful creation or termination of an instance.

Features Overview :
  • A template for successful creation / deletion.
  • Locate the error caused in which activity.
  • The error is displayed in a user understandable format.
  • The activites that take place during a creation / deletion
  • The logs are preprocessed and stored in .csv format.

GRAPHICAL STATISTICS

Provides a graphical visualization statistically for the comparison between successful creation or deletion and error creation or deletion.

Features Overview :
  • Displays Statistics of the creation / deletion.
  • Red bars are for error.
  • Green bars indicate successful launches
  • Comparison bar graphs are generated
  • The logs are preprocessed and stored in .csv format.

ABOUT PROJECT

An OpenStack setup usually consists of a very large pool of compute, storage and network logs. In this project, we propose to analyze various logs produced by different components of Openstack and also provide a visualization for an easier understanding. more..
