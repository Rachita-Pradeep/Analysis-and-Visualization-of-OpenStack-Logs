ABOUT PROJECT

Overview Of Our Project

An OpenStack setup usually consists of a very large pool of compute, storage and network logs. One of the biggest challenges for OpenStack administrators is to monitor stacks and analyze logs for anomalies. There is a need for a tool to analyze the logs and provide a clean visualization. In this project, we propose to analyze various logs produced by different components of Openstack and also provide a visualization for an easier understanding.

The project is about the Analysis and Visualization of Openstack logs, with a view to aid Software developers in debugging of Systems that use Openstack infrastructure used in various applications of cloud computing such as email and SMAC.

Future Scope

The future scope of this project is to analyze logs that are generated when Openstack is being used for application basis as well and not just to provide Infrastructure as a service. We can also do further research on ELK Stack and apply Machine learning Algorithms for analysis and try for a better accuracy.
